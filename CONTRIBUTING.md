# Contributing

Thanks for helping improve the Interactive Learning Tutor skill!

## How to contribute

### Reporting issues

If the tutor behaves unexpectedly — wrong mode transitions, poor Socratic questions, level anchoring that misses the mark — open an issue with:

1. What you said to trigger the skill
2. What the tutor did
3. What you expected it to do

### Proposing changes

This skill is a teaching protocol defined in `SKILL.md`. When proposing changes:

1. Explain the teaching scenario that isn't well-handled
2. Propose a concrete rule change or addition
3. Consider how it interacts with the existing state machine and the 10 behavior rules

### Pull requests

- Keep changes focused on `SKILL.md`
- Describe the pedagogical rationale in the PR description
- Test your changes by installing the modified skill locally and running a few learning sessions

## Code of Conduct

Be respectful. This is an educational tool — discussions should match that tone.
